运行项目

  • 在 gradle 任务中找到 my-project/Tasks/dev/Run, 双击运行
  • 运行结果如下:
......
> Task :my-project:t0300GoBuild
# go build -tags tmpl,static -o my-project.exe

> Task :my-project:Run
# my-project.exe run
http://localhost:8080

00:57:32: 执行完成 'Run'。
  • 点击 http://localhost:8080 会打开浏览器, 并访问项目
